📌 Project Overview

This project builds a smart recommendation engine that analyzes movie features and recommends similar or relevant movies to users. It can be extended into a full-fledged AI-powered streaming platform recommendation feature.

🚀 Features
🎯 Personalized movie recommendations
🔍 Search movies by name
📊 Filter based on genre, mood, language, rating, and year
🤖 Machine Learning-based similarity matching
📁 CSV dataset integration
⚡ Fast and lightweight system
