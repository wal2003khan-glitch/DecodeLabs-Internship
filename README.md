🤖 Simple Python Chatbot

A beginner-friendly AI chatbot built using Python with basic programming concepts like conditions, loops, and a knowledge-based response system. The chatbot can interact with users, answer simple questions, and simulate human conversation through predefined logic.

🚀 Features
💬 Interactive text-based conversation
🧠 Rule-based response system
🔁 Continuous chat loop
❓ Help command support
🚪 Exit command to close chatbot
🐍 Built completely in Python
